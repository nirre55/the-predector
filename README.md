# the-predector

> A machine learning prediction system built with Python, scikit-learn, LightGBM, and real-time data via WebSockets.

## Getting Started

### Prerequisites

- Python 3.10+
- pip or a virtual environment manager (e.g. `venv`, `conda`)

### Installation

```bash
# Clone the repo
git clone <repo-url>
cd the-predector

# Create and activate a virtual environment
python -m venv .venv
source .venv/bin/activate  # Windows: .venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt
```

### Usage

```bash
python main.py
```

## Running Tests

```bash
pytest tests/
```
